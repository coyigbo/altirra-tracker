## Altirra CRM Issue Tracker

This repository contains the public issue tracker for the Altirra CRM product.  
Customers and internal users can use GitHub Issues to report bugs, request features, and ask for support.

### How to Submit an Issue

1. Go to the **Issues** tab in this repository.
2. Click **New issue**.
3. Choose the template that best matches your request:
   - **Bug report** – for problems or defects in the CRM.
   - **Feature request** – for new functionality or improvements.
   - **Data or account issue** – for data quality, access, or account problems.
   - **Integration or API issue** – for sync or API-related problems.
   - **General support request** – for questions and non-urgent help.
4. Fill out all required fields with as much detail as possible.

### What Happens Next

- Our team reviews new issues regularly.
- We may follow up in the comments if we need more information.
- Status updates (in progress or complete) will be reflected via labels and/or comments on the issue.

### Security & Sensitive Information

- **Do not** share passwords, API keys, or other secrets in issues.
- If you need to share sensitive data, we will provide a secure channel separately.
